## About this repository

Cats is a tinder-like app for cats.

## Installation

First, copy the env.example file, rename it .env and fill it with your environement variables such as database credentials.

```
$ composer install && npm install
$ php artisan key:generate
$ php artisan migrate --seed
$ php artisan serve
$ npm run dev
or
$ npm run watch // for live reloading
